## 1. 프로젝트 개요

### 1.1 컨셉
**Kitchen Simulator**는 중식당 주방을 시뮬레이션하는 교육용 게임입니다. 플레이어는 실시간으로 들어오는 주문을 받아 정확한 레시피에 따라 조리하고 시간 내에 서빙해야 합니다.

### 1.2 타겟 사용자
- 요식업 신입 직원 교육
- 주방 업무 체험을 원하는 일반인
- 타임 매니지먼트 게임 애호가

### 1.3 핵심 가치
- **실제감**: 웍 온도, 재료 투입 시 온도 변화 등 실제 주방 물리 시뮬레이션
- **교육성**: 정확한 레시피 순서, 타이밍, 불 조절 학습
- **난이도 조절**: 신입부터 관리자까지 3단계 난이도

## 2. 핵심 기능

### 2.1 실시간 주문 시스템
```
┌─────────────────────────────────┐
│  ⏱️ 00:03:42  📋 주문 대기열  │
├─────────────────────────────────┤
│ 🍜 계란볶음밥    [화구1 배정]  │
│ ⏱️ 3:42                          │
├─────────────────────────────────┤
│ 🥘 마파두부      [화구2 배정]  │
│ ⏱️ 2:10                          │
└─────────────────────────────────┘
```

**기능**:
- 난이도별로 다른 주기로 주문 생성
  - 신입: 30초마다 1개
  - 알바: 20초마다 2개
  - 관리자: 15초마다 3개
- 실시간 타이머 표시 (각 주문별)
- 7분 이내 서빙 목표
- 15분 초과 시 자동 취소

### 2.2 다중 화구 관리
```
┌──────┐  ┌──────┐  ┌──────┐
│화구1 │  │화구2 │  │화구3 │
│ 🔥   │  │ 💧   │  │ 🍳   │
│180°C │  │ 25°C │  │320°C │
└──────┘  └──────┘  └──────┘
```

**기능**:
- 3개의 독립적인 화구
- 각 화구마다 개별 온도 관리
- 동시에 여러 메뉴 조리 가능
- 화구별 상태 추적 (깨끗함/젖음/더러움/과열/타버림)

### 2.3 정밀한 온도 시뮬레이션
**온도 구간**:
- **25°C**: 실온 (초기 상태)
- **180°C**: 볶기 가능 온도
- **300°C**: 스모킹 포인트 (연기 발생)
- **360°C**: 과열 경고
- **400°C**: 타버림 (조리 실패)

**온도 변화 요인**:
- **불 세기**: 약불(0.78배) / 중불(1.56배) / 강불(1.82배)
- **재료 투입**: 채소(-40°C), 해산물(-45°C), 계란(-20°C), 밥(-15°C)
- **조리 액션**: 볶기(-10°C), 뒤집기(-8°C)
- **물 넣기**: 온도 25°C로 리셋

### 2.4 레시피 검증 시스템
```
[계란볶음밥 레시피]
┌─────────────────────────────────┐
│ ① 재료 투입                     │
│   ☑ 식용유 10ml                 │
│   ☑ 계란 2개                    │
│   ⬜ 밥 200g                     │
├─────────────────────────────────┤
│ ② 액션: 볶기 (180°C 이상)      │
│   ⬜ 완료 대기                   │
└─────────────────────────────────┘
```

**검증 항목**:
- 재료 종류 (SKU 일치)
- 재료 양 (정확한 양)
- 조리 순서 (스텝 순서)
- 조리 타이밍 (제한 시간)

### 2.5 배치 투입 시스템
```
[여러 재료 한 번에 넣기]
           화구1   화구2   화구3
양파       50g     0g      30g
당근       30g     20g     0g
애호박     40g     0g      0g

        [확인]  [취소]
```

**기능**:
- 냉장고/서랍에서 여러 재료 동시 선택
- 각 화구에 개별적으로 양 지정
- 한 번에 여러 웍에 투입

## 3. 게임 모드

### 3.1 난이도별 차이

| 구분 | 신입 | 알바 | 관리자 |
|------|------|------|--------|
| **주문 주기** | 30초마다 | 20초마다 | 15초마다 |
| **동시 주문** | 1개 | 2개 | 3개 |
| **목표 완료** | 3개 | 10개 | 20개 |
| **실수 처리** | 즉시 차단 | 경고 후 진행 | 경고 후 진행 |
| **타이밍 오버** | 타버림 | 점수 감점 | 점수 감점 |

### 3.2 신입 모드 (BEGINNER)
**특징**:
- 가장 느린 주문 속도
- 틀린 재료 투입 시 즉시 차단
- 잘못된 액션 시 즉시 차단
- 타이밍 오버 시 웍이 타버림
- **목적**: 정확한 레시피 학습

### 3.3 알바/관리자 모드 (INTERMEDIATE/ADVANCED)
**특징**:
- 빠른 주문 속도
- 틀려도 진행 가능 (오류 누적)
- 서빙 시 정확도 표시
- 점수에 반영
- **목적**: 속도와 효율성 향상

## 4. 주방 레이아웃

### 4.1 전체 레이아웃
```
┌──────────────────────────────────────────────────────────┐
│                     🍜 주문 대기열                       │
├──────────┬──────────────────────────────┬───────────────┤
│          │                              │               │
│  싱크대  │         화구 3개             │   조미료대    │
│   🚰     │     🔥      🔥      🔥       │    🧂🧂      │
│          │                              │    🧂🧂      │
│          ├──────────────────────────────┤    🧂🧂      │
│          │                              │    🧂🧂      │
│ 4호박스  │      서랍 냉장고 (2x2)       │               │
│  ❄️❄️    │      🧊🧊                    │               │
│  ❄️❄️    │      🧊🧊                    │               │
│          │                              │               │
├──────────┴──────────────────────────────┴───────────────┤
│                   📋 레시피 가이드                       │
├──────────────────────────────────────────────────────────┤
│         📄 액션 로그        │    🔥 화구 사용률         │
└──────────────────────────────────────────────────────────┘
```

### 4.2 주방 구역 설명

#### 왼쪽: 싱크대 + 4호박스
- **싱크대**: 웍 씻기 (더러운 웍 → 젖은 웍)
- **4호박스**: 2x2 칸, 각 칸마다 1층/2층 선택 가능

#### 중앙: 화구 + 서랍 냉장고
- **화구 3개**: 메인 조리 공간
- **서랍 냉장고**: 2x2 서랍, 자주 쓰는 재료 보관

#### 오른쪽: 조미료대
- **4x2 배치**: 소금, 설탕, 간장, 식용유, 참기름 등

## 5. 조리 시스템

### 5.1 기본 조리 흐름
```
[주문 들어옴]
     ↓
[화구에 배정] → 웍이 자동으로 불 켜짐
     ↓
[재료 투입] → 냉장고/서랍/조미료대에서 재료 선택
     ↓
[양 입력] → 화구별로 양 지정
     ↓
[액션 실행] → 볶기/뒤집기/물넣기
     ↓
[온도 관리] → 불 세기 조절 (약/중/강)
     ↓
[서빙] → 점수 계산 후 완료
```

### 5.2 래디얼 메뉴 (Radial Menu)
웍을 클릭하면 주변에 버튼이 나타납니다:

```
        [볶기 🍳]
           ↑
[물넣기 💧] ← 웍 → [뒤집기 🔄]
           ↓
  [약불 중불 강불 🔥]
```

**기능**:
- 웍 클릭 시에만 표시
- 액션 실행 후 자동 닫힘
- ESC 키로 닫기
- 외부 클릭 시 닫기

### 5.3 웍 상태 관리

#### 깨끗함 (CLEAN) ✅
- **색상**: 밝은 회색
- **상태**: 조리 가능
- **조건**: 설거지 후 말린 상태 또는 180°C 도달

#### 젖음 (WET) 💧
- **색상**: 파란빛
- **상태**: 조리 불가 (말려야 함)
- **말리기**: 불 켜고 대기 → 180°C 도달 시 자동으로 CLEAN

#### 더러움 (DIRTY) 🧹
- **색상**: 갈색
- **상태**: 조리 불가 (씻어야 함)
- **처리**: 싱크대로 이동 → 설거지 → WET → 말림 → CLEAN

#### 과열 (OVERHEATING) ⚠️
- **색상**: 주황색, 경고 표시
- **온도**: 360~400°C
- **위험**: 온도 낮추지 않으면 타버림

#### 타버림 (BURNED) 🔥
- **색상**: 검은색
- **온도**: 400°C 이상
- **결과**: 조리 실패, 메뉴 취소, 설거지 필요

### 5.4 물 시스템
```
[물 넣기] → 온도 25°C로 리셋
     ↓
[가열] → 100°C까지 30초 소요
     ↓
[100°C 도달] → 타이머 시작
     ↓
[5초 유지] → 끓기 시작 (보글보글 💦)
```

**특징**:
- 물이 있으면 일반 온도 계산 중단
- 끓는점 유지 시 애니메이션 표시
- 국물 요리에 사용

## 6. 점수 시스템

### 6.1 시간 점수
```
완벽 (0~7분):     100점  ✅
양호 (7~10분):     85점  👍
느림 (10~15분):    70점  ⚠️
매우 느림 (15분~): 30점  ❌
자동 취소 (15분+): -50점 🚫
```

### 6.2 레시피 정확도 점수
```
정확도 100%: 100점
정확도 80%:  80점
정확도 60%:  60점
틀린 레시피: 30점 (시간 10~15분과 동일)
```

### 6.3 최종 점수
```
최종 점수 = (시간 점수 + 레시피 점수) / 2
```

**예시**:
- 6분 내 정확한 레시피: (100 + 100) / 2 = **100점**
- 8분, 오류 1개 (정확도 90%): (85 + 90) / 2 = **87.5점**
- 12분, 오류 3개 (정확도 70%): (70 + 70) / 2 = **70점**
- 16분 초과: 자동 취소, **-50점**

### 6.4 게임 점수
```
총점 = (레시피 정확도 50%) + (속도 30%) + (화구 효율 20%)
```

## 7. UI/UX 설명

### 7.1 색상 시스템

#### 온도 표시
- **파란색** (0~179°C): 안전
- **노란색** (180~299°C): 볶기 가능
- **주황색** (300~359°C): 스모킹 포인트
- **빨간색** (360~399°C): 과열 경고
- **검은색** (400°C+): 타버림

#### 타이머 색상
- **초록색** (~7분): 완벽
- **노란색** (~10분): 양호
- **주황색** (~15분): 경고
- **빨간색** (15분+): 치명적 (깜빡임)

### 7.2 애니메이션

#### 불 애니메이션 🔥
- **조건**: 볶기 중 + 온도 180°C 이상
- **지속 시간**: 1초
- **효과**: 웍질 시 불이 번쩍 (실제 주방 재현)

#### 연기 애니메이션 💨
- **조건**: 온도 300°C 이상 (스모킹 포인트)
- **효과**: 웍에서 연기 상승

#### 물 끓는 애니메이션 💦
- **조건**: 물 온도 100°C + 5초 유지
- **효과**: 보글보글 (물방울 아이콘 애니메이션)

#### 과열 애니메이션 ⚠️
- **조건**: 온도 360°C 이상
- **효과**: 웍 테두리 주황색 깜빡임

### 7.3 사운드 효과 (예정)
- 🔥 불 켜는 소리
- 🍳 볶는 소리
- 💦 물 끓는 소리
- ⚠️ 경고음 (과열, 타이밍 오버)
- ✅ 서빙 완료 알림
- ❌ 오류 소리

### 7.4 키보드 단축키
- **ESC**: 모든 팝업 닫기
- **Enter**: 양 입력 확인
- **Tab**: 입력 필드 이동
- **1/2/3**: 화구 선택 (예정)

## 8. 사용자 시나리오

### 8.1 신입 첫 플레이
```
1. 난이도 "신입" 선택
2. 첫 주문 "계란볶음밥" 들어옴
3. 화구1에 배정
4. 레시피 가이드 확인: "① 식용유 10ml"
5. 조미료대 클릭 → 식용유 선택
6. 양 입력: 화구1에 10ml
7. 투입 성공! ✅
8. 다음 단계: "② 계란 2개"
9. 서랍 냉장고 클릭 → 계란 선택
10. 양 입력: 화구1에 2개
11. 온도 확인: 150°C (아직 볶기 불가)
12. 불 세기 "강불"로 조절
13. 온도 상승: 180°C 도달
14. 웍 클릭 → 래디얼 메뉴
15. "볶기" 클릭 → 불 애니메이션 🔥
16. 모든 단계 완료
17. "서빙" 버튼 클릭
18. 점수 표시: "⏱️ 5분 32초 - 완벽! (100점)"
```

### 8.2 알바 빠른 플레이
```
1. 주문 2개 동시 들어옴: "짜장면", "짬뽕"
2. 짜장면 → 화구1, 짬뽕 → 화구2 배정
3. 서랍 냉장고 클릭
4. 양파, 돼지고기 선택 (다중 선택)
5. 양 입력 팝업:
   - 양파: 화구1에 50g, 화구2에 30g
   - 돼지고기: 화구1에 100g
6. 한 번에 투입
7. 화구1: 춘장 추가
8. 화구2: 해산물 추가
9. 동시 조리 진행...
10. 서빙 완료
```

## 10. FAQ

### Q1: 웍이 타버렸는데 어떻게 하나요?
**A**: 싱크대로 이동시켜 설거지하세요. 젖은 상태가 되면 불을 켜서 180°C까지 말려야 다시 사용할 수 있습니다.

### Q2: 재료를 잘못 넣었어요!
**A**: 
- **신입 모드**: 잘못된 재료는 투입되지 않습니다.
- **알바/관리자 모드**: 투입되지만 오류로 기록되고 점수가 감점됩니다.

### Q3: 온도가 너무 빨리 올라가요!
**A**: 불 세기를 "약불" 또는 "중불"로 조절하세요. 강불은 빠르지만 과열 위험이 있습니다.

### Q4: 메뉴가 15분 넘어서 사라졌어요!
**A**: 15분 초과 시 자동으로 주문이 취소됩니다. 더 빠르게 조리하거나, 웍을 비우고 다시 시작하세요.

### Q5: 여러 재료를 한 번에 넣고 싶어요!
**A**: 냉장고나 서랍에서 여러 재료를 선택한 후, 배치 입력 팝업에서 화구별로 양을 지정하세요.
