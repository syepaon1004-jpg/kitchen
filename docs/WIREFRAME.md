## 7. 접근성 (Accessibility)

### 7.1 키보드 단축키
```
- ESC: 모든 팝업 닫기
- Enter: 양 입력 확인
- Tab: 다음 입력 필드
- Shift+Tab: 이전 입력 필드
- 1/2/3: 화구 선택 (예정)
```

### 7.2 색상 대비
```
- 텍스트: 4.5:1 이상 (WCAG AA)
- 온도 게이지: 색맹 고려 (색상 + 숫자)
- 버튼: 명확한 테두리
```

### 7.3 포커스 표시
```
- 입력 필드: ring-2 ring-blue-500
- 버튼: outline-none ring-2 ring-offset-2
```
